PLATFORM USED TO BUILD: Linux (Ubuntu 22.04)

To build: g++ main.cpp Command.cpp Controller.cpp ECGraphicViewImp.cpp Model.cpp -I. -lallegro -lallegro_font -lallegro_primitives -lallegro_image -lallegro_main 

IMPORTANT : When saving to a file, you must hit ESCAPE key before closing!!!!!!!


All the features asked of us have been implemented:

Inserting rectangles and ellipses (filled or not)
Selecting shape or multiple shapes
Grouping shapes
Ungrouping shapes
Moving shapes with mouse and arrows
Removing shapes
Loading and saving to file (NOTE: MUST HIT ESCAPE TO SAVE FILE) !!!!!!!!!!
Undo / redo
Color can be added as well through editing the file
